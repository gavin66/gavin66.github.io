基于 laravel5.0(现已更新到5.2) 开发的博客系统

服务器需求

  • PHP版本 >= 5.5.9
  • Mysql 版本 >= 5.7 ( 数据存储用到了 JSON 数据类型)
  • 必须安装 Redis(使用它缓存数据)
  • composer (用来下载更新 laravel 依赖库)

安装

  1. 下载源码

  2. 安转 laravel 依赖库

  3. 修改配置文件,把文件 .env.example 重命名为 .env

  4. 生成应用 key

    将应用的 key（APP_KEY）设置为一个随机字符串,如果应用 key 没有被设置，用户 Session 和其它加密数据将会有安全隐患。
